🎯 Top Priority (Must-Have)
1. Intersection Observer API
Detect when elements enter/leave viewport
Perfect for: scroll animations, lazy loading, infinite scroll
You already have a page started: browser-apis/intersection-observer.astro
2. Web Animations API (WAAPI)
JavaScript equivalent of CSS animations with way more control
Perfect for: complex sequences, pause/play/reverse animations, dynamic values
Better performance than manipulating styles directly
This is THE animation API for modern web
3. Resize Observer API
Detect when elements change size
Perfect for: responsive animations, container queries behavior, adaptive layouts
Triggers before paint (no layout thrashing)
4. Scroll-Driven Animations API ⭐ NEW & EXCITING
CSS animations that respond to scroll position
Perfect for: parallax, scroll progress indicators, reveal animations
Still experimental but game-changing for scroll effects
🎨 Secondary Priority (Very Useful)
5. requestAnimationFrame (RAF)
Smooth 60fps animations
Perfect for: custom animations, game loops, smooth scrolling
Essential for performance
6. Pointer Events API
Unified touch/mouse/stylus events
Perfect for: drag interactions, drawing, gesture controls
Better than separate mouse/touch handlers
7. View Transitions API ⭐ NEW
Smooth page/state transitions with automatic animations
Perfect for: SPA navigation, dialog animations, image galleries
Chrome-first but coming to other browsers
🔧 Supporting APIs
8. Mutation Observer API
Watch for DOM changes
Perfect for: reacting to dynamic content, tracking user edits
Pairs well with animations when content changes
9. Page Visibility API
Detect when tab is visible/hidden
Perfect for: pausing animations when tab hidden (performance)
Simple but important for battery life
10. Performance API / PerformanceObserver
Measure animation performance, layout shifts
Perfect for: debugging jank, optimizing animations
